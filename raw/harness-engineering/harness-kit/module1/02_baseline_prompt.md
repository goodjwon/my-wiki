# 실습 1-2: 베이스라인 측정

## 목적
하네스를 적용하기 **전** 에이전트 성능을 측정합니다.
나중에 CLAUDE.md + Hooks 적용 후와 비교해서 실제 개선을 수치로 확인합니다.

---

## 베이스라인 측정 태스크 3종 (순서대로 실행)

### 태스크 A — 도메인 모델에 필드 추가

```
[하네스 없이 실행 - CLAUDE.md, hooks 없는 상태]

다음 요구사항을 구현해줘:

계약(Contract) 엔티티에 '계약 담당자 이메일(managerEmail)' 필드를 추가해야 해.
이메일 형식 검증이 필요하고, null 허용 안 됨.
관련 API도 업데이트해줘.
```

**측정 항목**:
- [ ] DTO 변환 없이 Entity 직접 반환했는가?
- [ ] Migration 파일 생성했는가?
- [ ] 테스트 작성했는가?
- [ ] 요청하지 않은 코드가 추가되었는가? (몇 줄?)
- [ ] 완료까지 주고받은 메시지 횟수:

