## 1. 변경

```python
class TestResult:
    def __init__(self):
        self.runCount = 0
    def summary(self):
        return f"{self.runCount} run, 0 failed"

class TestCase:
    def run(self, result):
        result.runCount += 1
        self.setUp()
        try:
            method = getattr(self, self.name)
            method()
        finally:
            self.tearDown()
```
