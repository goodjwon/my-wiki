## 1. 첫 번째 RED

```python
class TestCase:
    def __init__(self, name): self.name = name
    def run(self):
        method = getattr(self, self.name)
        method()

class WasRun(TestCase):
    def __init__(self, name):
        self.wasRun = None
        super().__init__(name)
    def testMethod(self):
        self.wasRun = 1

test = WasRun("testMethod")
print(test.wasRun)   # None
test.run()
print(test.wasRun)   # 1
```

→ "테스트 메서드가 호출되면 표시" — xUnit 의 가장 단순한 코어.
