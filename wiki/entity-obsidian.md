# Obsidian

로컬 마크다운 파일 기반의 지식 관리 앱.

## LLM Wiki에서의 역할

[[src-llm-wiki-pattern|LLM Wiki 패턴]]에서 Obsidian은 **위키 뷰어**로 기능한다. "Obsidian은 IDE, LLM은 프로그래머, 위키는 코드베이스."

## 유용한 기능

- **그래프 뷰** — 위키의 연결 구조를 시각화, 허브·고아 페이지 식별
- **[[entity-dataview|Dataview]]** — frontmatter 기반 동적 쿼리
- **[[entity-marp|Marp]] 플러그인** — 마크다운 슬라이드
- **[[entity-obsidian-web-clipper|Web Clipper]]** — 웹 기사를 마크다운으로 클리핑

## 설정 팁

- 첨부파일 경로를 `raw/assets/`로 고정하면 이미지를 로컬에서 관리 가능
- "Download attachments" 핫키 지정 → 클리핑 후 이미지 일괄 다운로드
