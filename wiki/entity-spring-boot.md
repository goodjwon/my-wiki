# Spring Boot

Spring 기반 독립 실행형 프로덕션급 애플리케이션을 최소 설정으로 만들 수 있는 Java 프레임워크.

## 핵심 특징

| 특징 | 설명 |
|------|------|
| 독립 실행 | 내장 서버(Tomcat/Jetty/Undertow)로 WAR 배포 불필요 |
| 자동 설정 | Spring + 서드파티 라이브러리 자동 구성 |
| Starter 의존성 | opinionated 의존성 묶음으로 빌드 단순화 |
| 프로덕션 기능 | 메트릭, 헬스체크, 외부화 설정 기본 제공 |
| 제로 코드젠 | 코드 생성·XML 설정 없음 |

## 버전

- 최신: **4.0.5** (2026-04-18 기준)
- 2.x → 3.x 마이그레이션 관련 참고 영상 존재
