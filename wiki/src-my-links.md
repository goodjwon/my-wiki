# 나의 링크

## 개요

Notion "나의 링크" 데이터베이스에 짬짬이 모아 둔 북마크 35건. 2021년부터 축적된 개발 관련 참고 링크 모음이다.

## 주제별 분류

### Java / Spring
- Spring Boot 쓰레드 풀 사용 (ThreadPoolTaskExecutor)
- Java 8 람다 표현식, 함수형 인터페이스
- Stream max/min 예제
- JPA Entity → DTO 매핑 (MapStruct)
- Spring Modulith
- FixtureMonkey 테스트 픽스처
- 테스트 피라미드 번역 (The Practical Test Pyramid)

### 프론트엔드 / TypeScript
- TypeScript & Node.js 디자인 패턴
- Python vs JavaScript 비교
- CSS Grid, Paper CSS
- Figma to Code (OROR Forge) 제작기

### 아키텍처 / 조직
- 기술 부채 유형과 추적 방법
- IT 프로젝트 실패 8가지 이유
- 개발자가 "No"라고 말할 때 질문 4가지
- 데일리 스크럼 개선 (컬리 기술 블로그)

### 기타
- OpenAI Structured Outputs
- AWS Workshop Studio
- IoT 전자저울 → Excel 연동
- KT홈허브 + ipTIME 멀티브릿지

## 활용 방법

위키 페이지 작성 시 참고자료로 연결하거나, 특정 주제 심화 학습 시 출발점으로 사용한다.
