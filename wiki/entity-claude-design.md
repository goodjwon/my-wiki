# Claude Design

Anthropic이 출시한 AI 디자인 도구. 디자인 시스템 생성, 와이어프레이밍, 프로토타이핑, 슬라이드덱 제작을 하나의 환경에서 수행할 수 있다.

## 주요 특징

- **디자인 시스템 자동 생성**: 회사 정보, GitHub, Figma 등을 입력하면 컴포넌트 라이브러리 생성
- **의도 파악 대화**: 프롬프트 입력 시 질문을 통해 사용자 의도를 정확히 파악 (클로드 채팅과 동일한 방식)
- **다중 시안 비교**: 여러 변형을 생성하고 변수(레이아웃, 밀도, 색상 등)별로 비교 가능
- **인터랙티브 프로토타입**: 버튼, 다이얼로그 등이 실제로 작동하는 프로토타입 출력
- **Edit / Comment 모드**: 직접 수정 또는 프롬프트 기반 부분 수정
- **Claude Code 핸드오프**: 디자인 시스템과 레퍼런스를 Claude Code에 전달하여 바로 구현 연계
- **파일 시스템 기반**: 생성물이 디자인 파일로 관리됨 (HTML 형태)

## 제작 모드

| 모드 | 용도 |
|------|------|
| 프로토타이핑 | 와이어프레임 → 디자인 → 인터랙티브 프로토타입 |
| 슬라이드덱 | 프레젠테이션 자료 제작 |

## 내보내기

ZIP, PDF, PPTX, Canva, Claude Code 핸드오프

## 포지셔닝

기존 AI 디자인 도구(Stitch, Paper, Pencil)와 유사하나, Anthropic 생태계 내에서 **디자인 → 코드** 파이프라인을 단일 컨텍스트로 완결시킨다는 점이 차별점. 2026년 4월 기준 리서치 프리뷰 상태.
