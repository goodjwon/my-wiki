# Obsidian Web Clipper

웹 기사를 마크다운으로 변환하여 [[entity-obsidian|Obsidian]] vault에 저장하는 브라우저 확장 프로그램.

## LLM Wiki에서의 역할

`raw/`에 소스를 빠르게 추가하는 주요 수단. 웹에서 기사를 발견하면 클리핑 → raw/에 저장 → [[concept-ingest|Ingest]] 워크플로 실행.

## 관련

- [[entity-obsidian|Obsidian]] — 호스트 앱
- [[concept-ingest|Ingest]] — 클리핑 후 실행하는 워크플로
