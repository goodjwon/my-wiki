# Dataview

[[entity-obsidian|Obsidian]] 플러그인. YAML frontmatter를 기반으로 동적 테이블·리스트를 생성하는 쿼리 엔진.

## LLM Wiki에서의 활용

위키 페이지에 frontmatter(tags, dates, sources 등)를 일관되게 달아두면 Dataview로 동적 뷰를 만들 수 있다:
- 특정 태그를 가진 페이지 목록
- 최근 업데이트된 페이지
- 소스별 관련 위키 페이지
